3. HTML是由<strong>标签</strong>构成的
4. `<title></title>`用于<strong>页面</strong>标题
5. `<h1></h1>`用于标题，最大h6  
PS：<h7>这个是h7，无意义</h7>
6. `<p></p>`是段落
7. `<strong></strong>`用于<strong>加粗</strong>
8. `<em></em>`用于<em>斜体</em>  
PS：经典梗：`<em>本田</em>`

`&lt;` 小于
